Turtle

For my python turtle assignment I used PyGLM to create a spinning cube that bounces around. PyGLM is a library that adds GLM math functions to python so that I could use matrix multiplication and vectors to get the coordinates of the cube so that I could just use the turtle to draw. There is an issue where for whatever reason it uses an orthographic projection as opposed to perspective, which makes distance not really work in this. But it's not really visible in this particular demo.
